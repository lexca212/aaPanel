# Third-Party Plugin: Security Tools

This plugin adds quick actions for:
- SSH log
- System log
- Open `/var/log`
- Fail2Ban status
- Install/Enable Fail2Ban
- Disable Fail2Ban

## Build ZIP package
```bash
./third_plugins/build_security_tools_zip.sh
```
Output:
`third_plugins/dist/security_tools_plugin.zip`

## Install in aaPanel (App Store)
1. Open **App Store**
2. Go to **Third-party**
3. Click **Import plugins**
4. Upload `security_tools_plugin.zip`
5. Confirm install

## Open plugin
After installed, open plugin **Security Tools** from plugin list.
